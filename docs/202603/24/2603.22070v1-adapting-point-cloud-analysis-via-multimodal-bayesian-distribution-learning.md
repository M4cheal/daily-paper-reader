## 摘要
多模态3D视觉-语言模型在多种3D任务中表现出强大的泛化能力，但在领域偏移下其性能仍会显著下降。这激发了近期对测试时适配（TTA）的研究，该技术使模型能够利用测试时数据进行在线适配。在现有的TTA方法中，基于缓存的机制被广泛采用，用于在在线预测优化中利用先前观察到的样本。然而，它们仅存储有限的历史信息，导致随着测试流的演进，信息逐渐丢失。此外，它们的预测逻辑值（logits）采用启发式融合，使得适配过程不稳定。为了解决这些局限性，我们提出了 BayesMM，一种用于测试时点云分析的多模态贝叶斯分布学习框架。BayesMM 将每个类别的文本先验和流式视觉特征建模为高斯分布：文本参数源自语义提示，而视觉参数则随到达的样本进行在线更新。这两种模态通过贝叶斯模型平均进行融合，该方法根据后验证据自动调整它们的贡献，从而产生一个无需训练即可持续适配不断演进的测试时数据的统一预测。在多个点云基准测试上的广泛实验表明，BayesMM 在分布偏移下保持了鲁棒性，平均性能提升超过 4%。

## Abstract
Multimodal 3D vision-language models show strong generalization across diverse 3D tasks, but their performance still degrades notably under domain shifts. This has motivated recent studies on test-time adaptation (TTA), which enables models to adapt online using test-time data. Among existing TTA methods, cache-based mechanisms are widely adopted for leveraging previously observed samples in online prediction refinement. However, they store only limited historical information, leading to progressive information loss as the test stream evolves. In addition, their prediction logits are fused heuristically, making adaptation unstable. To address these limitations, we propose BayesMM, a Multimodal Bayesian Distribution Learning framework for test-time point cloud analysis. BayesMM models textual priors and streaming visual features of each class as Gaussian distributions: textual parameters are derived from semantic prompts, while visual parameters are updated online with arriving samples. The two modalities are fused via Bayesian model averaging, which automatically adjusts their contributions based on posterior evidence, yielding a unified prediction that adapts continually to evolving test-time data without training. Extensive experiments on multiple point cloud benchmarks demonstrate that BayesMM maintains robustness under distributional shifts, yielding over 4% average improvement.

## 1. 核心问题与研究背景
*   **核心问题**：大型多模态 3D 视觉-语言模型（如 ULIP, Uni3D）虽然在零样本任务中表现出色，但在面对测试环境与训练环境不一致（领域偏移，Domain Shift）时，性能会显著下降。
*   **研究背景**：现有的测试时自适应（TTA）方法主要依赖“缓存机制”（Cache-based），即存储高置信度的测试样本特征。然而，这种方式存在两大缺陷：
    1.  **信息丢失**：缓存容量有限，随着测试流的推进，旧的有用信息会被覆盖。
    2.  **融合不稳定**：多模态（文本与视觉）的融合通常采用启发式的超参数调节，缺乏理论支撑且在不同领域间表现不稳定。

## 2. 方法论：BayesMM 框架
*   **核心思想**：将测试时自适应建模为“多模态贝叶斯分布学习”，通过维护类别的统计分布（高斯分布）而非离散样本，实现连续、稳健的在线适配。
*   **关键技术细节**：
    *   **文本分布学习（Textual Distribution Learning）**：利用大语言模型（LLM）生成类别的多种同义描述（Paraphrases），计算其嵌入向量的均值和协方差，构建文本先验分布，捕捉语义多样性。
    *   **几何分布学习（Geometric Distribution Learning）**：将流式输入的点云特征视为观测值。利用贝叶斯递归更新规则，在线更新每个类别的几何分布参数（均值 $\mu$ 和协方差 $\Sigma$）。
    *   **贝叶斯模型平均（Bayesian Model Averaging, BMA）**：这是融合的核心。模型不使用固定的权重，而是根据当前样本在两种模态下的后验证据（Evidence），动态计算文本和几何模态的贡献权重，生成最终预测。
*   **算法流程**：输入点云 $\rightarrow$ 提取特征 $\rightarrow$ 递归更新几何分布参数 $\rightarrow$ 计算文本与几何模态的后验概率 $\rightarrow$ 贝叶斯加权融合 $\rightarrow$ 输出类别。

*   **性能领先**：BayesMM 在所有基准测试中均优于现有的缓存方法，平均准确率提升超过 **4%**，在某些严重腐蚀场景下提升更显著。

*   **模态偏好**：实验发现，在领域偏移下，文本模态通常比几何模态更稳定，贝叶斯框架能自动识别并赋予文本模态更高的权重。

## 7. 优点与亮点
*   **无需训练（Training-free）**：不需要访问源域数据，不需要更新骨干网络参数，部署成本极低。
*   **理论支撑**：用贝叶斯概率框架替代了经验性的启发式融合，解决了 TTA 过程中的不确定性问题。
*   **扩展性强**：内存开销随类别数增长极慢，非常适合处理具有成千上万个类别的开放世界识别任务。

## 8. 不足与局限
*   **高斯分布假设**：该方法假设特征空间服从高斯分布。虽然在深度特征空间中这通常是合理的，但在某些极端复杂的非线性分布下，高斯建模可能不够精确。
*   **推理延迟**：虽然不需要训练，但相比原始的零样本推理，由于涉及分布更新和贝叶斯计算，吞吐量有微小下降（约 3%-5%）。
*   **依赖预训练质量**：作为一种适配方法，其性能上限仍受限于原始 3D 基础模型的特征提取能力。
