## 摘要
视觉基于模型的强化学习（MBRL）智能体在训练分布上表现良好，但一旦测试环境发生偏移，往往会失效。在视觉 MBRL 中，识别偏移的发生通常较为容易；难点在于将这种识别转化为有效的动作级修正。我们研究了应对偏移的几种方法，包括规划惩罚、直接微调、全局残差修正和粗粒度门控。在实验中，这些方法要么无法改善闭环控制，要么会损害分布内（ID）的性能。基于这些负面结果，我们提出了 JEPA 索引的局部专家增长（JEPA-Indexed Local Expert Growth）。该方法仅将冻结的 JEPA 表示用于问题索引，而特定于簇的残差专家在原始控制器的基础上添加局部动作修正。基准控制器本身不被修改。通过配对自助法（paired-bootstrap）评估，我们发现原始的朴素偏好变体在更严格的测试下并不稳定。相比之下，硬配对（harder-pair）变体在所有四种评估的偏移条件下都产生了显著的分布外（OOD）改进，同时保持了 ID 性能。当再次遇到相同的偏移时，学习到的专家仍然有效，这支持了将适应视为增量知识增长而非重复全量重训的观点。我们进一步表明，利用简单的密度模型可以实现自动 ID 拒绝，而 OOD 子族之间的细粒度区分则受限于表示。总体而言，结果表明，对于分布偏移下的视觉 MBRL，主要挑战不仅在于察觉环境发生了变化，而是在识别出变化后应用正确的局部动作修正。

## Abstract
Visual model-based reinforcement learning (MBRL) agents can perform well on the training distribution, but often break down once the test environment shifts. In visual MBRL, recognizing that a shift has occurred is often the easier part; the harder part is turning that recognition into useful action-level correction. We study several ways of responding to shift, including planning penalties, direct fine-tuning, global residual correction, and coarse gating. In our experiments, these approaches either do not improve closed-loop control or hurt in-distribution (ID) performance. Based on these negative results, we propose JEPA-Indexed Local Expert Growth. The method uses a frozen JEPA representation only for problem indexing, while cluster-specific residual experts add local action corrections on top of the original controller. The baseline controller itself is not modified. Using paired-bootstrap evaluation, we find that the original naive-preference variant is not stable under stricter testing. In contrast, the harder-pair variant produces statistically significant OOD improvements on all four evaluated shift conditions while preserving ID performance. The learned experts also remain useful when the same shift is encountered again, which supports the view of adaptation as incremental knowledge growth rather than repeated full retraining. We further show that automatic ID rejection can be achieved with simple density models, whereas fine-grained discrimination among OOD sub-families is limited by the representation. Overall, the results indicate that, for visual MBRL under distribution shift, the main challenge is not simply noticing that the environment has changed, but applying the right local action correction after the change has been recognized.

### 2. 方法论：JEPA-Indexed Local Expert Growth
论文提出了一种模块化的适应框架，核心思想是“保持基准控制器不变，仅在需要时挂载局部专家”。
*   **核心思想**：将“识别问题”与“解决问题”解耦。使用冻结的 JEPA 编码器作为“索引器”，识别当前属于哪种偏移场景，然后调用对应的“残差专家”进行动作微调。
*   **关键技术细节**：
    1.  **问题索引（Indexing）**：使用冻结的 JEPA 提取观察序列的特征，通过质心算法（Centroid-based）将当前状态归类为特定的偏移簇（如躯干质量增加 3 倍）或拒绝为 ID 状态。
    2.  **局部残差专家（Local Residual Experts）**：为每个已知的偏移簇训练一个小型专家网络 $E_c$。最终动作 $a_t = a_{base} + \Delta a^{(c)}_t$，其中 $a_{base}$ 是原始控制器的输出，$\Delta a$ 是专家提供的修正量。
    3.  **偏好学习（Preference-based Training）**：专家不通过模仿学习训练，而是通过“硬配对（Harder-pair）”挖掘，学习将基准动作推向更高回报动作的方向。
    4.  **ID 拒绝机制**：引入密度模型，当检测到状态属于 ID 分布时，不激活任何专家，确保原始性能。

### 6. 主要结论与发现
*   **检测易，适应难**：简单的距离信号足以检测 OOD，但不足以指导动作。
*   **模块化优势**：局部专家增长能在显著提升 OOD 性能（如在质量偏移下提升回报）的同时，几乎不损害 ID 性能（p > 0.05，无显著差异）。
*   **适应即增长**：证明了专家在“第二次相遇”时依然有效，支持了将适应视为知识增量积累的观点。
*   **表示瓶颈**：发现目前的 JEPA 表示足以区分 ID 和 OOD，但难以在无监督下区分不同的 OOD 子类（如 3x 质量和 5x 质量在潜空间高度重叠）。
*   **适用边界**：该方法在基准模型失效模式一致（低方差）时效果最好；若偏移导致基准模型表现出高方差（如齿轮比偏移），则难以学习到有效的修正。

### 7. 优点（亮点）
*   **架构解耦**：不修改原始控制器，通过残差方式挂载专家，极大地缓解了灾难性遗忘问题。
*   **统计严谨性**：采用 30 种子配对评估，这在强化学习论文中属于较高标准的实验诚信。
*   **现实意义**：提出了“二次相遇”的概念，符合工业机器人多次面对同类故障/环境变化的实际场景。
*   **深入的失效分析**：不仅展示成功案例，还详细讨论了表示空间的局限性和方法失效的边界。

### 8. 不足与局限
*   **依赖先验**：目前仍需要手动定义偏移簇的路由规则，未能实现完全自动化的新偏移发现与聚类。
*   **任务覆盖面**：主要集中在 DMControl 的两个运动任务上，尚未在更复杂的视觉操作任务或真实机器人上验证。
*   **表示限制**：实验表明冻结的 JEPA 表示在区分细微物理参数变化上存在瓶颈，限制了自动路由的精度。
*   **专家泛化性**：一个偏移专家通常只能处理特定类型的偏移，缺乏跨偏移种类的泛化能力。
