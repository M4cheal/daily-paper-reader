## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变体以及人类纠错机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略开始，通过利用在机器人车队中收集的自主展开（rollouts）和人类干预，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，以便在基于流（flow-based）的 VLA 动作生成器中进行策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和时长 3-5 分钟的长程任务。随着车队经验的积累，单个通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

*   **研究目标**：建立一个“数据飞轮”，通过机器人集群（Fleet）在实际任务中产生的自主尝试、失败恢复和人类干预数据，持续优化共享的通用策略，使其具备更强的鲁棒性和处理长程任务（Long-horizon）的能力。

### 2. 方法论：核心思想与关键技术
LWD 提出了一个**离线到在线（Offline-to-Online）的强化学习框架**，包含两个关键算法组件：

*   **分布隐式价值学习（DIVL, Distributional Implicit Value Learning）**：
    *   **核心思想**：改进了 IQL（隐式 Q 学习）。由于集群数据极其异构（包含成功、失败、干预），传统的标量价值函数会模糊掉稀疏但关键的成功信号。
    *   **技术细节**：DIVL 学习动作价值的**分布**（Categorical Distribution），并提取特定分位数（Quantile）作为 TD 目标。
    *   **自适应策略**：引入基于分布熵的自适应分位数参数 $\tau$，在不确定性高时保持保守，在信心足时保持乐观。
*   **伴随匹配 Q 学习（QAM, Q-learning with Adjoint Matching）**：
    *   **核心思想**：解决流匹配（Flow Matching）策略的优化难题。直接对多步去噪过程进行反向传播既不稳定又耗时。
    *   **技术细节**：QAM 将 Critic 提供的动作梯度转化为流模型每一步的局部回归目标，实现了对大型生成式 VLA 策略的高效、稳定更新。
*   **训练流程**：分为离线预训练（利用 652 小时混合数据）和在线持续后训练（利用集群实时回传的混合 Buffer）。

*   **实验场景**：在 16 台 Agibot G1 双臂机器人上进行了 8 项真实任务测试。

    *   **长程任务（3-5 分钟）**：功夫茶冲泡、鸡尾酒调制、果汁制作、鞋盒打包（侧重多阶段规划与恢复）。
*   **对比基准（Baselines）**：
    *   **SFT**：仅使用演示数据的监督微调。
    *   **RECAP**：一种基于优势加权的离线后训练方法。
    *   **HG-DAgger**：交互式模仿学习，利用人类干预数据。

### 6. 主要结论与发现
*   **性能飞跃**：LWD 在所有任务上的平均成功率达到 **95%**，显著优于 SFT (76%) 和 RECAP (85%)。
*   **长程任务优势**：在长程任务中，LWD 的提升最为显著（从 SFT 的 0.68 提升到 0.91），证明了 TD 误差反向传播在多步决策中的有效性。
*   **效率提升**：相比 SFT，LWD 训练出的策略动作更果断，平均作业周期缩短了约 23.75 秒。
*   **分布式的价值**：分布价值函数（DIVL）比标量函数在处理异构集群数据时更稳健，尤其在离线到在线切换时表现更好。

### 7. 优点（亮点）
*   **系统性创新**：成功闭合了“部署-收集-训练-再部署”的闭环，是目前少有的能在大规模真实机器人集群上运行的在线 RL 系统。
*   **算法适配性**：DIVL + QAM 的组合巧妙解决了 VLA 模型在 RL 训练中的不稳定性问题。
*   **真实性**：完全基于真实物理经验，不依赖模拟器，处理了包括人类干预在内的复杂数据流。

### 8. 不足与局限
*   **更新频率**：目前的在线更新采用简单的实时调度，对于更大规模（如成千上万台）的部署，如何保持更新的稳定性和效率仍需探索。
*   **推理与规划限制**：长程任务仍依赖简单的语言指令，对于需要复杂逻辑推理或多轮对话纠错的场景，VLA 的推理能力仍有提升空间。
*   **安全性建模**：框架目前未显式加入安全约束（Safety-aware learning），在处理具有潜在危险的部署环境时可能存在风险。
