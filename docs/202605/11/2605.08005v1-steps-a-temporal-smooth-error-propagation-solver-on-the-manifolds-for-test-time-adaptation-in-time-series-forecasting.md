## 摘要
测试时自适应（TTA）旨在利用推理过程中揭示的有限观测值，来改进分布偏移下的时间序列预测。然而，预测任务中的 TTA 必须在无源在线环境下运行，此时自适应信号较短、具有时间相关性且可能存在噪声。因此，当揭示的前缀（prefix）稀疏或受污染时，现有方法可能会面临可辨识性弱、误差累积以及长程修正不稳定等问题。为了解决这些问题，我们提出了 STEPS，一种用于时间序列预测 TTA 的平滑时间误差传播求解器。STEPS 将预测 TTA 重新表述为时间流形上的狄利克雷边界值问题（Dirichlet Boundary Value Problem），其中揭示的前缀误差作为未知未来误差场的边界条件。随后，STEPS 在预测空间中求解一个平滑且有界的修正场：局部求解器（Local Solver）在时间平滑性约束下传播前缀误差，全局求解器（Global Solver）检索稳定的跨窗口误差记忆，而时空流形融合（SMF）将这两种解集成到最终的修正中。在六个标准基准数据集和四个冻结的主干网络上，STEPS 相比零样本（zero-shot）主干网络实现了 26.82% 的平均相对 MSE 降低，比最强的对比 TTA 基准方法高出 12.77%。额外的稀疏前缀和污染测试证实了 STEPS 在有限且含噪前缀下的鲁棒性。

## Abstract
Test-Time Adaptation (TTA) aims to improve time series forecasting under distribution shifts by using limited observations revealed during inference. However, forecasting TTA must operate in a source-free online setting, where the adaptation signal is short, temporally correlated, and potentially noisy. Existing methods can therefore suffer from weak identifiability, error accumulation, and unstable long-horizon corrections when the revealed prefix is sparse or contaminated. To address these issues, we propose STEPS, a Smooth Temporal Error Propagation Solver for TTA in time-series forecasting. STEPS reformulates forecasting TTA as a Dirichlet Boundary Value Problem on a temporal manifold, where the revealed prefix error serves as the boundary condition for the unknown future error field. Then, STEPS solves a smooth and bounded correction field in prediction space: a Local Solver propagates prefix errors under temporal smoothness, a Global Solver retrieves stable cross-window error memory and Spatiotemporal Manifold Fusion (SMF) integrates both solutions into the final correction. Across six standard benchmarks and four frozen backbones, STEPS achieves an average relative MSE reduction of 26.82% over the zero-shot backbone, exceeding the strongest compared TTA baseline by 12.77%. Additional sparse prefix and contamination tests confirm the robustness of STEPS under limited and noisy prefixes.

### 1. 核心问题与研究动机
在实际应用中，时间序列预测模型常面临**分布偏移**（如季节性变化、传感器老化、市场波动等），导致预训练模型在推理阶段表现下降。**测试时自适应（TTA）**旨在利用推理时实时观测到的少量数据（前缀数据）来修正预测。
*   **核心挑战**：现有的预测 TTA 方法通常将前缀数据视为在线微调的标签，但在前缀数据**稀疏**（观测点极少）或**受污染**（存在离群点/噪声）时，容易导致模型过度拟合噪声、误差累积或长程预测轨迹崩溃。
*   **研究目标**：提出一种更稳健的机制，将前缀误差视为约束而非简单的训练标签，实现平滑且有界的预测修正。

### 2. 方法论：STEPS 框架
STEPS 的核心思想是将预测 TTA 重新表述为**时间流形上的狄利克雷边值问题（Dirichlet Boundary Value Problem）**。它不修改主干网络参数，而是直接在输出空间求解修正场。

*   **前缀边界选择**：利用快速傅里叶变换（FFT）估计主导周期，动态确定前缀长度，将观测到的残差作为边界条件。
*   **局部求解器（Local Solver）**：基于**平滑误差流形假设**，通过最小化离散狄利克雷能量（即惩罚相邻步长间的剧烈波动），利用调和传播（Harmonic Propagation）将边界误差平滑地延伸至未来。
*   **全局求解器（Global Solver）**：维护一个**动态误差记忆库**，存储历史窗口的完整误差模板。通过一个轻量级神经解码器，根据当前预测和记忆检索长程误差模式（如持续性偏差或相位偏移）。
*   **空时流形融合（SMF）**：使用一个随时间步增长的 S 型函数（Ramp function）进行融合。近场预测主要依赖局部平滑传播，远场预测则更多参考全局记忆。最后通过**显式幅度裁剪（Clipping）**确保修正场不会因异常值而发散。

*   **性能领先**：STEPS 在所有设置下均显著优于 Baseline，平均 MSE 降低 **26.82%**，比最强竞争对手 COSA 领先约 12.77 个百分点。

*   **稀疏有效性**：即使只有 2-3 个观测点，STEPS 也能通过平滑传播提供有效的近场修正，而不会导致远场预测漂移。

### 7. 优点与亮点
*   **数学严谨性**：将 TTA 引入狄利克雷边值问题的视角，为误差修正提供了坚实的几何与物理直觉。
*   **即插即用**：无需重新训练或微调主干网络，对计算资源要求极低，非常适合边缘设备部署。
*   **稳定性设计**：通过 SMF 融合和显式裁剪，解决了 TTA 中常见的“修正过度”和“噪声放大”痛点。

*   **记忆依赖性**：全局求解器的效果依赖于历史误差模式的重复性。如果发生剧烈的、前所未有的突发性制度变更（Regime Change），记忆库可能失效。
