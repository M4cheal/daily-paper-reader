## 摘要
世界动作模型 (WAMs) 通过联合预测场景演变和机器人动作来增强视觉-语言-动作 (VLA) 策略，但现有方法通常将预测的世界表示为整体图像、视频 token 或全局潜变量。当指令涉及特定对象时，动作解码器难以寻址这些表示，尤其是在场景偏移导致对象身份与上下文纠缠的情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操控的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位 (slot) 状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并与文本、图像、本体感受和过去动作的 token 在块因果序列中融合。世界头预测下一帧的槽位状态，而流匹配 (flow-matching) 动作头在同一次前向传播中解码 16 步连续动作块。通过仅使用地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，强制实现可寻址性，从而在不增加额外 token 的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试得出的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）在标准基准测试中表现优异，但在面对场景扰动（如摄像头视角切换、物体布局改变、光照变化）时极易崩溃。
*   **研究动机**：作者认为这种不稳定性源于“对象寻址能力（Object Addressability）”的缺失。传统模型将场景视为整体图像或全局 token，导致目标物体的身份与周围环境上下文纠缠在一起。
*   **核心问题**：当指令要求“拿起红杯子”时，如果场景发生几何偏移，模型往往无法在全局表示中稳定定位该物体。论文旨在构建一种能将“操作哪个物体”与“物体当前状态”在张量层面解耦的模型。

#### 2. 论文提出的方法论
OA-WAM 的核心思想是将世界建模为一组**可寻址的对象槽位（Object Slots）**。
*   **槽位分解**：每一帧被分解为 $N+1$ 个槽位（1 个机器人槽位，$N$ 个物体槽位）。每个槽位向量由两部分组成：
    *   **地址向量 (`addr`)**：基于语言标签和初始特征计算，在整个回合中冻结，代表物体身份。
    *   **内容向量 (`content`)**：随时间变化，捕捉物体的姿态、外观和状态。
*   **关键技术细节**：
    *   **地址唯一键路由**：在 Transformer 的每一层，跨槽位注意力的“键（Key）”投影仅读取 `addr` 子向量。
    *   **残差重置挂钩**：在每层残差连接后，强制将 `addr` 部分重置为初始值，防止身份信息在深层网络中被环境噪声污染。
    *   **统一序列建模**：使用 7B 参数的 Chameleon 风格主干网络，融合文本、图像 VQ、本体感受、槽位状态和过去动作。
    *   **双头预测**：世界头（World Head）回归下一帧槽位状态；动作头（Action Head）利用流匹配（Flow-matching）一次性解码 16 步连续动作块。

#### 3. 实验设计
*   **数据集/场景**：
    *   **LIBERO**：评估分布内（In-distribution）的成功率，涵盖空间、物体、目标和长程任务。
    *   **SimplerEnv (WidowX/Bridge)**：评估模拟到真实的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性测试，包含摄像头、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动轴。
*   **对比方法**：对比了包括 OpenVLA、$\pi_0$、$\pi_{0.5}$、InternVLA-M1 等顶级 VLA 模型，以及 WorldVLA、VLA-JEPA、Cosmos-Policy 等世界模型基准。

#### 5. 实验数量与充分性
*   **实验规模**：在三个主流基准上进行了详尽测试，每个任务均运行 3 个随机种子并取平均值。
*   **消融实验**：设计了 A1（对象可寻址性隔离）、A2（因果槽位干预）、A3（世界头作用）等多组消融，验证了地址掩码和重置机制的必要性。
*   **客观性**：通过“交换绑定（Swap-binding）”测试，客观量化了模型是否真的在利用地址空间进行决策，而非依赖视觉记忆。实验设计严谨，对比基准涵盖了 2024-2025 年间几乎所有主流模型。

#### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超过了 SOTA 水平。
*   **几何鲁棒性**：在 LIBERO-Plus 的几何扰动轴（摄像头/机器人初始态/布局）上，OA-WAM 显著优于所有基准，几何平均成功率比 $\pi_{0.5}$ 高出 4.8%。
*   **机制验证**：因果干预显示 OA-WAM 的物体绑定余弦相似度高达 0.87，而传统整体式模型均低于 0.09，证明了其架构确实实现了对象级解耦。

#### 7. 优点
*   **架构创新**：通过简单的张量掩码和重置机制，在不增加额外 token 的情况下实现了强力的对象寻址约束。
*   **可验证性**：不同于黑盒 VLA 模型，OA-WAM 的决策逻辑（即关注哪个槽位）是架构上可解释且可干预的。
*   **鲁棒性强**：有效解决了 VLA 模型在视角切换和布局改变时容易“跟丢”目标的问题。

#### 8. 不足与局限
*   **感知瓶颈**：模型高度依赖上游的感知组件（SAM 3, DINOv3），在传感器噪声轴上表现较弱（-17.1%），因为图像模糊会直接导致槽位提取失败。
*   **仿真局限**：所有验证均在模拟器中完成，尚未展示在真实物理机器人上的部署效果。
*   **计算开销**：感知前端（SAM 3 等）的延迟（138ms）远高于策略网络本身，限制了实时高频控制。
*   **假设限制**：模型假设物体间是弱耦合的，在处理物体间复杂物理相互作用（如堆叠、遮挡）时可能存在挑战。
