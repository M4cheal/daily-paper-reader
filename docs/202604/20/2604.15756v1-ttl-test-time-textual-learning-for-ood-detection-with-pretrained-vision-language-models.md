## 摘要
视觉-语言模型（VLM）如 CLIP 通过对齐视觉和文本表示，展现出强大的分布外（OOD）检测能力。近期基于 CLIP 的测试时自适应方法通过引入外部 OOD 标签进一步提升了检测性能。然而，此类标签是有限且固定的，而真实的 OOD 语义空间本质上是开放的。因此，固定标签无法表征测试流中遇到的多样化且不断演变的 OOD 语义。为了解决这一局限性，我们提出了测试时文本学习（TTL），这是一个无需依赖外部 OOD 标签，即可从无标签测试流中动态学习 OOD 文本语义的框架。TTL 利用伪标签测试样本更新可学习提示（prompts），以捕获新出现的 OOD 知识。为了抑制伪标签引入的噪声，我们提出了一种 OOD 知识净化策略，在抑制噪声的同时选择可靠的 OOD 样本进行自适应。此外，TTL 维护了一个存储高质量文本特征的 OOD 文本知识库，为跨批次的数据提供稳定的分数校准。在两个标准基准和九个 OOD 数据集上的广泛实验表明，TTL 持续达到最先进的性能，突显了文本自适应对于鲁棒的测试时 OOD 检测的价值。我们的代码可在 https://github.com/figec/TTL 获取。

## Abstract
Vision-language models (VLMs) such as CLIP exhibit strong Out-of-distribution (OOD) detection capabilities by aligning visual and textual representations. Recent CLIP-based test-time adaptation methods further improve detection performance by incorporating external OOD labels. However, such labels are finite and fixed, while the real OOD semantic space is inherently open-ended. Consequently, fixed labels fail to represent the diverse and evolving OOD semantics encountered in test streams. To address this limitation, we introduce Test-time Textual Learning (TTL), a framework that dynamically learns OOD textual semantics from unlabeled test streams, without relying on external OOD labels. TTL updates learnable prompts using pseudo-labeled test samples to capture emerging OOD knowledge. To suppress noise introduced by pseudo-labels, we introduce an OOD knowledge purification strategy that selects reliable OOD samples for adaptation while suppressing noise. In addition, TTL maintains an OOD Textual Knowledge Bank that stores high-quality textual features, providing stable score calibration across batches. Extensive experiments on two standard benchmarks with nine OOD datasets demonstrate that TTL consistently achieves state-of-the-art performance, highlighting the value of textual adaptation for robust test-time OOD detection. Our code is available at https://github.com/figec/TTL.

*   **研究动机**：深度学习模型在开放世界中经常会将分布外（OOD）样本错误地识别为高置信度的分布内（ID）类别。虽然视觉-语言模型（VLM，如CLIP）提升了检测能力，但现有的测试时自适应（TTA）方法通常依赖于**固定且有限的外部OOD标签**。
*   **核心矛盾**：现实中的OOD语义空间是无限且动态演变的，固定的标签无法涵盖测试流中出现的各种未知语义。
*   **研究目标**：提出一种无需外部标签、能从无标签测试流中动态学习OOD文本语义的框架。

### 2. 方法论
TTL框架主要包含三个核心组件，旨在通过测试时自适应优化文本提示（Prompts）：
*   **OOD知识学习（OOD Knowledge Learning）**：
    *   为每个ID类别引入可学习的OOD提示词。
    *   利用基础检测器生成的伪标签，通过“OOD关注少数平衡损失”（$L_{OMB}$）来优化提示词的前缀，使OOD提示词与伪标签为OOD的图像特征对齐。
*   **OOD知识净化（OOD Knowledge Purification, OKP）**：
    *   **核心思想**：解决伪标签中的噪声问题（即被误标为OOD的ID边界样本）。
    *   **技术细节**：利用自适应阈值将伪OOD样本分为高置信度子集和低置信度（边界）子集。通过损失函数 $L_{OKP}$ 增大高置信度样本与OOD提示词的相似度，同时抑制边界样本的影响。
*   **OOD文本知识库（OOD Textual Knowledge Bank, OKB）**：
    *   动态存储高质量的OOD文本特征，容量固定为 $K$。
    *   **更新策略**：优先保留与ID文本特征距离最远的（即最具辨别力的）OOD特征。
*   **分数校准**：在推理时，将基础检测器的得分与知识库计算出的校准得分进行加权融合，以增强对OOD数据的抑制。

### 6. 主要结论与发现
*   **性能领先**：TTL 在 ImageNet-1k 上平均 FPR95 降低了 **12.67%**，AUROC 提升了 **3.94%**，显著优于现有的 TTA 方法。
*   **文本学习的价值**：直接从测试流学习文本语义比单纯适配视觉特征或使用固定标签更有效。
*   **净化机制的关键性**：OKP 策略能有效分离 ID 边界样本，防止模型在自适应过程中发生语义漂移。

### 7. 优点
*   **无需先验**：完全摆脱了对外部 OOD 标签或数据集的依赖，更符合实际部署场景。
*   **动态演进**：能够捕捉测试流中新出现的、未见的 OOD 语义。
*   **可解释性**：通过可视化 OKB 中的近邻词汇，证明了模型确实学习到了与 OOD 领域相关的语义（如自然景观、特定材质等）。
*   **兼容性强**：可以作为插件集成到现有的多种 OOD 检测器中。

### 8. 不足与局限
*   **计算开销**：由于在测试阶段需要进行梯度更新（Prompt Tuning），其推理延迟高于无需更新的事后方法。
*   **基础检测器依赖**：虽然 TTL 具有鲁棒性，但如果基础检测器的初始信号完全是随机噪声（如极低质量的伪标签），自适应的效果会受到限制。
*   **存储需求**：虽然 4MB 很小，但在极端嵌入式设备上仍需考虑知识库的内存管理。
