## Abstract
This paper explores the weakly-supervised referring image segmentation (WRIS) problem, and focuses on a challenging setup where target localization is learned directly from image-text pairs. 
We note that the input text description typically already contains detailed information on how to localize the target object, and we also observe that humans often follow a step-by-step comprehension process (\ie, progressively utilizing target-related attributes and relations as cues) to identify the target object. 
Hence, we propose a novel Progressive Comprehension Network (PCNet) to leverage target-related textual cues from the input description for progressively localizing the target object.
Specifically, we first use a Large Language Model (LLM) to decompose the input text description into short phrases. These short phrases are taken as target-related cues and fed into a Conditional Referring Module (CRM) in multiple stages, to allow updating the referring text embedding and enhance the response map for target localization in a multi-stage manner.
Based on the CRM, we then propose a Region-aware Shrinking (RaS) loss to constrain the visual localization to be conducted progressively in a coarse-to-fine manner across different stages.
Finally, we introduce an Instance-aware Disambiguation (IaD) loss to suppress instance localization ambiguity by differentiating overlapping response maps generated by different referring texts on the same image. 
Extensive experiments show that our method outperforms SOTA methods on three common benchmarks.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究问题**：弱监督指代图像分割（Weakly-Supervised Referring Image Segmentation, WRIS），即在仅有图像-文本对作为监督（无像素级分割掩码）的情况下，学习根据自然语言描述定位并分割目标物体。
- **动机**：现有WRIS方法（如TRIS、SAG）将整个描述编码为单一语言嵌入，容易忽略关键属性或关系线索，导致定位模糊甚至错误。受人类逐步理解过程（先定位所有相关物体，再通过附加属性逐步缩小范围）启发，论文提出一种渐进式理解网络（PCNet），通过逐步融合文本中的目标相关线索来提升定位精度。
- **整体含义**：该方法显著降低了RIS任务对精细标注的依赖，证明了仅通过文本监督即可达到接近全监督的性能，具有实际应用价值（如减少标注成本）。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：模仿人类逐步理解过程，利用大语言模型（LLM）将输入描述分解为多个短短语（目标相关线索），然后通过多阶段条件引用模块（CRM）逐步更新文本嵌入并生成响应图，实现从粗到精的定位。
- **关键技术细节**：
  - **目标相关线索生成**：使用LLM（Mistral 7B）通过精心设计的提示（包含通用指令、输出约束、上下文示例和输入问题）将原始描述分解为K个短短语（标准化为5个，不足则重复补全）。
  - **条件引用模块（CRM）**：由两个子模块组成：
    - **引用调制模块**：在每个阶段n，使用当前短语特征qn和视觉特征Vn，通过视觉到文本交叉注意力获取视觉关注后的短语特征，再通过文本到文本交叉注意力更新全局引用嵌入Qn → Qn+1，强化目标相关特征。
    - **响应图生成**：通过文本到视觉交叉注意力更新视觉特征，并计算更新后的视觉特征与文本嵌入的余弦相似度，生成响应图Rn ∈ [0,1]。
  - **区域感知收缩损失（RaS Loss）**：利用预训练分割器（FreeSOLO或SAM）生成掩码提议，通过计算响应图与提议的对齐得分分离前景/背景区域，定义定位模糊度Samb_n，并约束后一阶段模糊度不大于前一阶段，实现背景区域逐步衰减、前景区域逐步聚焦。
  - **实例感知消歧损失（IaD Loss）**：针对同一图像中不同描述可能激活重叠区域的问题，额外采样Nd个不同目标的描述，计算它们的响应图，通过约束各响应图的最大对齐得分索引互不相同（使用one-hot + stop-gradient技巧），迫使不同描述激活不同实例。
- **训练总体损失**：L_total = L_Cls + L_RaS + L_IaD，其中L_Cls为分类损失（区分正负文本-图像对）。

## 6. 论文的主要结论与发现

- 提出的PCNet在三个基准上均显著优于现有WRIS方法，尤其在PointM指标上提升明显（例如RefCOCOg Val(G)上PointM达57.9%，比TRIS的52.5%高5.4个百分点）。
- 结合SAM作为提议生成器（PCNet_S）可获得更优性能，证明精确的定位点足以驱动强大分割器产生高质量掩码。
- 渐进式引入文本线索（CRM）和区域收缩损失（RaS）是关键贡献，实例消歧损失（IaD）进一步缓解同一图像多描述重叠问题。
- 全监督方法LAVT在Oracle条件下（使用GT定位点）性能可被超越，暗示弱监督RIS在定位点准确时能达到全监督水平。

## 7. 优点：方法或实验设计上的亮点

- **方法创新**：
  - 首次在WRIS中引入LLM进行文本分解，将人类逐步理解过程建模为多阶段条件调制，思路新颖且符合认知规律。
  - RaS损失通过前景/背景区域分离和模糊度约束实现渐进式聚焦，巧妙利用预训练分割器提供的掩码提议作为空间先验。
  - IaD损失针对同一图像多描述的歧义问题，采用硬分配和索引分离策略，简单有效。
- **实验设计亮点**：
  - 同时使用FreeSOLO（无监督）和SAM（有监督但无语义标签）两种提议生成器，验证方法对提议质量不敏感。
  - 不仅报告mIoU，还强调PointM指标，切合WRIS任务本质（定位是核心，分割可由现成模型完成）。
  - 详细消融各组件、阶段数量、调制策略等，提供深入见解。
